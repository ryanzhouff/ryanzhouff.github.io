Pycharm debug mode

Ryan Zhou
2023-12-22
debug,pycharm

设置好断点之后，程序执行到断点处，程序被挂起，暂停执行后面的代码

控制台左侧按钮：

  • 1-Return debug：重新运行 debug
  • 2-Resume Program：跳转到下一个断点，若无断点，则直接执行完程序
  • 3-Stop：停止 debug，关闭运行的程序
  • 4-View breakpoints：显示所有断点
  • 5-Mute breakpoints：让断点失效

控制台调试按钮：

  • 1-Show Execution Point：跳转到代码当前执行位置
  • 2-Step Over(F8)：单步执行，不进入函数内部，但函数会执行。在不存在函数的情况下和 Step Into 效果一样。
  • 3-Step Into(F7)：单步执行，遇到子函数会进入函数继续单步执行，有的会跳到源代码里面去执行
  • 4-Step Into My Code(Alt+Shift+F7)：单步执行，遇到子函数则进入并继续单步执行，不会进入到源代码中
  • 5-Step Out(Shift+F8)：跳出当前函数，返回到调用此函数的位置
  • 6-Run to Cursor：运行到光标位置
  • 7-Evaluate Expression：计算表达式

在计算表达式框里，可以变量、参数、方法，例如在if语句中查看表达式的值是 true 还是 false

断点条件设置，在断点上右键，添加 condition 即可
