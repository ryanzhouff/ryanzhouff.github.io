数据结构之数组

Ryan Zhou
2023-12-22
stack,queue,数组,数据结构

栈

push入栈，pop出栈。

  • push 可以是接收多个参数，返回最新的数组长度
  • pop 删除数组的最后一项，并返回被删除的项

队列

push 入队列，shift 出队列，删除数组第一项并返回它。

反向队列，pop出队列，unshift入队列，在数组开头添加多个值
