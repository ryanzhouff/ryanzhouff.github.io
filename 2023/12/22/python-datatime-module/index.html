Python datetime模块

Ryan Zhou
2023-12-22
datetime, python

datetime 是内置模块，用来处理日期和时间

datetime 模块提供了 5 个常用类：

  • date
  • time
  • datetime
  • timedelta
  • tzinfo

date 类

date 类包含三个参数，分别是 year，month，day，返回格式为 year-month-day

time 类

常用参数：hour, minute, second,

datetime 参数

datetime 可以看作 date 和 time 的合体

实例属性和方法

timedelta 类

表示两个 datetime 对象之间的时间差，即两个日期或日期时间之间的差，常用 days, seconds, microseconds

tzinfo 类

表示时区
Python 提供 zoneinfo 模块，指定时区
