基于Antd Spin组件的全屏loading

Ryan Zhou
2023-11-24
antd,spin,loading,react

使用 element-plus 的 Loading 组件的 service 制作全屏 loading 非常方便

但是 antd 的 Spin 组件并没有提供类似的 service API，需要一些 hack 方法来达到全屏 loading 的效果。在 ts 文件中(注意：不是tsx文件)，是无法直接使用 React element 的写法的，如<component/>，只能通过React.createElement()`方法创建组件。

首先创建一个包裹元素作为 Spin 组件的父元素，卸载组件时直接从 DOM 上删除父元素即可

这里有一个缺陷，Spin 组件的tip属性必须使用在 nest pattern 中，所以直接在 props 中使用tip属性控制台会提示：

而且通过报错信息定位到相关代码发现，fullscreen属性和tip不能同时存在

有两种解决办法：

  1. 不添加tip属性，即，只有 loading 图片，没有文字 ☑️
  2. 不设置fullscreen属性，然后通过自定义 CSS 控制 loading 的 sytle
